# IndexTTS PySide6 UI 使用说明

## 简介

这是一个使用PySide6开发的IndexTTS图形用户界面，提供了以下功能：

- 通过参考音频生成语音
- 角色管理（保存、加载、删除、导入、导出角色）
- 长文本自动分段处理
- 历史音频查看与试听

## 安装依赖

在使用UI之前，请确保已安装所有依赖：

```bash
pip install -r requirements.txt
```

如果出现兼容性错误，请运行`修复并运行.bat`脚本。

## 启动方式

双击`运行UI.bat`文件启动UI界面，或使用命令行：

```bash
python ui.py
```

## 使用说明

### 界面概览

界面分为上下两个区域：
- 上半部分：主要操作区域（角色管理、参考音频、推理文本、推理结果）
- 下半部分：历史结果音频列表

### 角色管理

角色管理功能允许您保存常用的参考音频，以便快速加载。每个角色都将音频数据一起打包为单一文件，方便分享：

1. **添加新角色**：
   - 选择参考音频
   - 点击「保存/更新」按钮
   - 输入角色名称并确认

2. **加载角色**：
   - 从下拉列表选择角色
   - 点击「加载」按钮

3. **删除角色**：
   - 从下拉列表选择角色
   - 点击「删除」按钮并确认

4. **导出角色**：
   - 从下拉列表选择角色
   - 点击「导出」按钮
   - 选择导出位置和文件名
   - 导出的角色文件包含所有必要的音频数据

5. **导入角色**：
   - 点击「导入」按钮
   - 选择之前导出的角色文件
   - 导入后即可使用该角色

### 语音生成

1. **选择参考音频**：
   - 从角色列表加载，或
   - 点击「选择参考音频」按钮从文件系统选择

2. **输入文本**：
   - 在文本框中输入要转换的文本
   - 文本过长会自动分段处理（约500字一段）

3. **生成语音**：
   - 点击「生成语音」按钮
   - 处理过程中状态栏会显示进度信息
   - 完成后结果会自动播放并添加到历史列表

### 历史记录

历史记录区域显示之前生成的所有音频文件：

- 点击列表中的项目可以在「推理结果」播放器中试听
- 点击「刷新列表」按钮可以更新历史记录

## 技术说明

1. **角色存储**：
   - 角色数据（包括音频二进制数据）打包存储在单一pickle文件中
   - 加载角色时，会将音频数据临时解压到磁盘
   - 程序退出时自动清理临时文件

2. **角色分享**：
   - 导出的角色文件是完整的单一文件，可以直接分享给其他人
   - 导入时会进行格式验证，确保文件完整性

## 注意事项

1. 首次使用时需要加载模型，可能需要一些时间
2. 文本过长时会自动分段处理，处理时间会相应增加
3. 所有生成的音频文件都保存在outputs目录下
4. 导入角色时如有同名角色，将会提示冲突

## 故障排除

如果遇到问题：

1. 确保已正确安装所有依赖（包括typing-extensions）
2. 确保checkpoints目录中有正确的模型文件
3. 检查状态栏和错误提示信息
4. 如果仍有问题，请查看控制台输出获取更详细的错误信息
